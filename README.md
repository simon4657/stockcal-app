# StockCal - 股市戰情室 📈

> AI 驅動的台股與美股事件日曆應用程式

## 🌟 特色功能

- **📅 事件日曆**：追蹤重要的財報、經濟數據與市場事件
- **🔥 每日熱點**：即時追蹤市場最熱門的族群與個股
- **🎯 操作策略**：每日更新的投資策略建議
- **🤖 Gemini AI**：深度事件分析與歷史回測
- **📱 響應式設計**：支援桌面與行動裝置

## 🚀 技術棧

- **前端框架**：React 19 + TypeScript
- **樣式**：Tailwind CSS 4
- **圖示**：Lucide React
- **建置工具**：Vite 7
- **AI 整合**：Google Gemini API

## 📦 安裝與執行

```bash
# 安裝依賴
pnpm install

# 開發模式
pnpm dev

# 建置生產版本
pnpm build

# 預覽建置結果
pnpm preview
```
